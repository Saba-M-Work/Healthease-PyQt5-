# Healthease - Doctor Booking Platform

A high-fidelity patient management desktop application built with **Python** and **PyQt5**. This project focuses on a logic-driven UI/UX to streamline physician discovery and appointment scheduling.

## 🚀 Core Features

* **Modular Architecture:** Utilizes a `QStackedWidget` system for seamless, state-based navigation.
* **Advanced Discovery:** Real-time doctor filtering by department, specialty, and name.
* **Dynamic Scheduling:** Interactive "Which time works best?" UI with 30-minute interval availability tracking.
* **Emergency Protocol:** Dedicated priority booking flow for urgent medical consultations.
* **Professional UI:** Modern, clinical aesthetic using custom QSS (Qt Style Sheets) and shadow effects.

---

## 🛠️ Tech Stack

* **Language:** Python 3.x
* **GUI Framework:** PyQt5
* **Logic:** Custom scheduling algorithms with conflict prevention.
